# Jada Platform - Protótipo Visual

Plataforma de cotação onde compradores publicam necessidades e vendedores enviam propostas.

## 🎨 Sobre o Projeto

Este é um protótipo visual completo (não funcional) para apresentação ao cliente. Todas as páginas, componentes, modais e formulários estão implementados visualmente, sem integração com backend ou banco de dados.

## 🚀 Tecnologias

- Next.js 14
- TypeScript
- TailwindCSS
- React Icons
- Lucide React

## 📦 Instalação

```bash
npm install
```

## 🏃 Executar

```bash
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## 📁 Estrutura

- `/app` - Páginas e rotas
- `/components` - Componentes reutilizáveis
- `/lib` - Utilitários e helpers
- `/styles` - Estilos globais

## 🎯 Funcionalidades Visuais

- ✅ Landing Page
- ✅ Página de Planos
- ✅ Login e Cadastro
- ✅ Dashboard Comprador
- ✅ Dashboard Vendedor
- ✅ Dashboard Admin
- ✅ Gestão de Usuários
- ✅ Criação de Requisições
- ✅ Envio de Propostas
- ✅ Modais e Popups
- ✅ Formulários com Validação Visual
